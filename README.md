## üõ†Ô∏è Tools Overview

### AI & Machine Learning
- **Ollama**: Run large language models locally on your Android device
- **TensorFlow**: Open-source machine learning framework
- **PyTorch**: Deep learning framework with tensor computation
- **Hugging Face Transformers**: Pre-trained models for NLP tasks

### Network Security & Analysis
- **Nmap**: Network discovery and security auditing
- **Wireshark**: Network protocol analyzer
- **Tcpdump**: Command-line packet analyzer
- **Aircrack-ng**: WiFi security assessment tools
- **Metasploit**: Penetration testing framework

### RF & SDR Tools
- **GNU Radio**: Signal processing toolkit
- **RTL-SDR**: Software defined radio with Realtek RTL2832U
- **HackRF**: Open source SDR platform
- **Gqrx**: Software defined radio receiver

### Automation & Monitoring
- **Cron**: Job scheduler for automated tasks
- **HTTP Shortcuts**: HTTP requests via widgets and shortcuts
- **Termux:API**: Android integration for Termux
- **Notification scripts**: Custom alerts and notifications
